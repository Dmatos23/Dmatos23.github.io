Introducción rápida a XML

¿Qué es XML?

XML (eXtensible Markup Language) es un lenguaje de marcado diseñado para almacenar y transportar datos de forma estructurada. A diferencia de HTML, XML se centra en describir el contenido (datos) y no en cómo se va a visualizar.

La principal idea de XML es permitir que aplicaciones diferentes puedan intercambiar información usando una estructura de datos clara y legible tanto para humanos como para máquinas.

Características principales de XML

  • Es extensible: tú defines tus propias etiquetas.
  • Es legible: su estructura es clara y fácil de entender.
  • Es independiente de plataforma y lenguaje de programación.
  • Se usa para almacenar, transportar e intercambiar datos.

Usos comunes de XML

  • Intercambio de datos entre sistemas y aplicaciones.
  • Archivos de configuración (por ejemplo, en aplicaciones web o móviles).
  • Servicios web (SOAP, algunos APIs antiguos, etc.).
  • Almacenar información estructurada, como catálogos de productos.

Ventajas y desventajas

  • Ventajas: estándar, estructurado, flexible y muy usado en integración.
  • Desventajas: puede ser verboso, pesa más que formatos como JSON y puede ser más complejo de procesar.

Ejemplo sencillo de XML

A continuación se muestra un ejemplo de archivo XML donde se define un catálogo de productos (podría ser algo similar a productos de DMStore): 

<?xml version="1.0" encoding="UTF-8"?>
<catalogo>
  <producto>
    <id>P001</id>
    <nombre>Perfume DM Intense</nombre>
    <categoria>Perfumería</categoria>
    <precio>3500.00</precio>
    <stock>25</stock>
  </producto>

  <producto>
    <id>T010</id>
    <nombre>Audífonos Bluetooth Pro</nombre>
    <categoria>Tecnología</categoria>
    <precio>4200.00</precio>
    <stock>10</stock>
  </producto>
</catalogo>

En este ejemplo:

  • <catalogo> es el elemento raíz que agrupa todos los productos.
  • Cada <producto> representa un ítem con sus datos (id, nombre, categoría, precio, stock).
  • Este archivo se puede usar para que diferentes sistemas lean los productos y sus características.

¿Por qué es útil XML?

XML es útil cuando necesitas:

  • Un formato estándar para compartir datos entre aplicaciones o empresas.
  • Estructurar información que pueda ser validada (por ejemplo, usando esquemas XSD).
  • Almacenar configuraciones que sean fáciles de editar y entender.
Tip: Puedes combinar XML con otros lenguajes (como PHP, C#, JavaScript, etc.) para leer y procesar estos datos y mostrarlos luego en una página web.
